# pie断点

- 假设程序的加载基地址是 0x55555000，而相对基址偏移是 0x1234（ida查看），那么 $rebase(0x1234) 将计算出断点应设置的绝对地址 0x55556234，然后可以使用 b *0x55556234 命令在该地址处设置断点。

# 关闭aslr
查看aslr配置：
```bash
cat /proc/sys/kernel/randomize_va_space
```

关闭aslr：
```bash
echo 0 > /proc/sys/kernel/randomize_va_space
```
